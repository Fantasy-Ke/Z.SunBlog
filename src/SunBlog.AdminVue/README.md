## Build Setup 

``` 
 # clone
git clone https://github.com/Fantasy-Ke/sunblog-vue.git
```

```
# cd
cd  sunblog-vue
```

```
# yarn dependencies
yarn
```

```
# Compiles and hot-reloads for development
yarn run dev
```

```
# Compiles and minifies for production
yarn run build
```
